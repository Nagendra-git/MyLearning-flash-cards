System design is the process of designing the architecture, component, database and communication between different parts of a software system so that it satisfies functional and non-functional requrements like scalability, reliability, performance, and maintanability.

  1. Understand the problem
  2. Identify the scope of the system
  3. Research and analyze existing systems
  4. Create a high-level design
  5. Refine the design
  6. Document the design
  7. Continuously monitor and improve the system
