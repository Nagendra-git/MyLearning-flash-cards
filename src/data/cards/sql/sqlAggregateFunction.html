SQL aggregate function

An SQL aggregate function performs a calculation on a set of values from multiple rows and returns a single, summarized value. These functions are essential for data analysis and are heavily used alongside the GROUP BY clause to categorize data summaries. Except for COUNT(*), all standard aggregate functions completely ignore NULL values during their calculation. 

The 5 Standard Aggregate FunctionsThe most frequently used SQL aggregate functions 
available across almost all databases like MySQL, PostgreSQL, and SQL Server include:
    ✅COUNT():  Returns the total number of rows or non-NULL values.
    ✅SUM():    Calculates the total combined sum of a numeric column.
    ✅AVG():    Computes the mathematical average (mean) of a numeric column.
    ✅MIN():    Finds the smallest (minimum) value in a column.
    ✅MAX():    Finds the largest (maximum) value in a column.
