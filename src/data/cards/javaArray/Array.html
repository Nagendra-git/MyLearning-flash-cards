Array

An array stores element in continous memory locations

    For example:
        int[] are = {10,20,30,40,50};
    Each int occupies 4 bytes.

Must Remember:
    ✅ Arrays are objects stored on the heap.
    ✅ All elements are stored one after another.
    ✅ Addres = Base + (Index * Element Size).
    ✅ offset = Index * Element Size.
    ✅ Size is fixed.
    ✅ arr.length is a field.
    ✅ Index starts at 0 because the first element has an offset of 0 bytes.
    ✅ Random access is O(1).
    ✅ Insertion and deletion in the middle require shifting elements, making them O(n).
    ✅ Arrays have a fixed size after creation.
