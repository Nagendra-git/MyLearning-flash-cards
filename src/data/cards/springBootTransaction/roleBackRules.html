Rollback Rules

    If any exception happens it will roleback.
    But which exceptions triggers Rollback?
    Default Rule:
        Spring roll backs for only for upchecked exceptions.
        ✅ Rollback
            RuntimeException
            NullPointerException
            ArithmeticException
            IllegalArgumentException
        No Rollback(Checked Exceptions)
            IOException
            SQLException
            Exception
        Unless configured.

rollbackFor and noRollbackFor

    rollbackFor:
        To rollback on checked exceptions
    Example:
        @Transactional(
            rollbackFor = IOException.class
        )
        public void saveOrder() throws IOException{
            orderService.saveOrder();
            throw new IOException();
        }
        It will roll back.

    noRollbackFor:
        Sometimes, you don't want rollback.
        @Transactional(
            noRollbackFor = RuntimeException.class
        )
        // Transaction ->COMMIT
