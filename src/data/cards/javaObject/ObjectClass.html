Java Object Class

    The Object class is the root(parent) of all classes in java.
    Every class in java directly or indirectly inherits from Object.

    Why we need the Object class?
    It provides common methods that every java object can use.
    Example:
        1.toString()
        2.equals()
        3.hashCode()
        4.clone()
        5.getClass()
        6.wait()
        7.notify()
        8.notifyAll()
