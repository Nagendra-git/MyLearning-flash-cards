toString()

Returns the String representation of an Object.

    Default Implimentations:
    Employee emp = new Employee();
    sysout(emp);//Employee@5acf9800
    Internally:
        sysout(emp); -> sysout(emp.toString());

Override toString()

    class Employee{
        int id =101;
        String name = "Nagendra";
        @Override
        public String toString(){
            return id+" "+name;
        }
    }
    o/p :- 101 Nagendra

    🎯 Interview Takeaway:
        To provide a meaningful representation of an object instead of the default ClassName@hashCode.
