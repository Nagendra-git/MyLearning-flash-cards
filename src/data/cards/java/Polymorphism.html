    Achived by Method Overloading.

Run-Time Polymorphism

    Parent reference referring to child object.
    Acchived by method overriding.
Java code
Method overriding:
        class Animal{
            void sound(){
                sysout("Animal make sound");
            }
        }
        class Dog extends Animal{
            @Override
            void sound(){
                sysout("Dog barks");
            }
        }
        class Main(){
            main(){
                Animal a = new Dog();//upcasting
                a.sound();
            }
        }
