Encapsulation means hiding data and providing controlled access to it through getter and setter methods.

Java code
Encapsulation:
        class Student{
            private String name;
            public getName(){
                return name;
            }
            public setName(String name){
                this.name = name;
            }
        }
        class Main{
            main(){
                Student std = new Student();
                std.setName("Nagendra");
                println(std.getName());
            }
        }
