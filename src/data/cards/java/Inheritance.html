Acquiring properties & behaviors is known as inheritance. There are 5 types of inheritance.

  1. Single Inheritance
    2. Pseudo code
    Single Inheritance:
            class Animal
                method eat()
                    print "Animal is eating"

            class Dog extends Animal
                main()
                    Dog d = new Dog()
                    d.eat()
  2. Multilevel Inheritance
    3. Pseudo code
    Multilevel Inheritance:
            class Animal
                method eat()

            class Dog extends Animal
                method bark()

            class Puppy extends Dog
                main()
                Puppy p = new Puppy()
                p.eat()
                p.bark()
  3. Hierarchical Inheritance
    4. Pseudo Code
    Hierarchical Inheritance:
            class Animal
                method eat()

            class Dog extends Animal

            class Cat extends Animal
                main()
                Dog d = new Dog()
                Cat c = new Cat()
                d.eat()
                c.eat()
  4. Multiple Inheritance (through interfaces)
    5. Pseudo code
    Multiple Inheritance (Using Interfaces):
            interface Fly
                method fly()

            interface Swim
                method swim()

            class Duck implements Fly, Swim
                main()
                Duck d = new Duck()
                d.fly()
                d.swim()
  5. Hybrid Inheritance
Pseudo Code
Hybrid Inheritance:
    class Animal

    interface Fly

    class Bird extends Animal implements Fly
        main()
        Bird b = new Bird()
