Association represents a relationship between two independent classes.

Aggregation(Weak HAS-A Realtionship)

Aggregation in java represents a weak HAS-A Realtionship where one class contains a reference of another class.

This is Weak HAS-A Realtionship beacuse child object is created outside the parent object

Java code
Aggregation:
        class Employee{
            int id;
            Address address;//Aggregation
        } 
        class Address{
            int id;
        }
        
        class Mian{
            main(){
                Address add = new Address(100);//outside
                Employee emp = new Employee(200, add);
            }
        }

Composition(Strong HAS-A Realtionship)

Composition in java represents a Strong HAS-A Realtionship where one class contains a reference of another class.

This is Strong HAS-A Realtionship beacuse child object is created inside the parent object

Java code
Aggregation:
        class Employee{
            int id;
            Address address;
            Employee(int id){
                this.id = id,
                this.address = new Address(100)//inside
            }
        }  
        class Mian{
            main(){
                Employee emp = new Employee(200);
            }
        }
