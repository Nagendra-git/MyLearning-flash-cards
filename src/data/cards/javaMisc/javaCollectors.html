Java Collectors

    In Java, the Stream API provides a modern way to process data.
    But after filtering, mapping and transforming, we usually need to gather results into a collection,
    a map or even a single value.
    Thats where Collectors come in.
        A Collector is an object that defines how to accumulate elements of a stream into a final result.   
        The Collectors utility class provides ready-to-use static methods, so you rarely need to write your own collectors.

Methods

    1.toList():
        Collect elements into a List.
        return List<T>
        
List<String>>list = names.stream().collect(Collectors.toList());
        
    2. toSet():
        Collect unique elements.
        return Set<T>
        
Set<String>set = names.stream().collect(Collectors.toSet());
        
    3. toCollection():
        Collect into a specific collection implementations.
        return : Collection you specify.
        
LinkedList<String> list =names.stream().collect(Collectors.toCollection(LinkedList::new));
        
    4. joining():
        Join Strings.
        return : String
        
String result =names.stream().collect(Collectors.joining(", ")); //o/p : John, David, Alice
        
    5. counting():
        count element.
        return: Long
        
Long count =names.stream().collect(Collectors.counting());
        
    6. summingInt(): // summingLong() //summingDouble()
        Sum  values.
        return : Integer
        
int total =employees.stream().collect(Collectors.summingInt(Employee::getSalary));
        
    7. averagingInt()// averagingLong()//averagingDouble()
        Calculate Average.
        
double avg = employees.steam().collect(Collectors.averagingDouble(Employee::getSalary));
        
    8. maxBy():
        Find maximum Object.
        return : Optional≤T>
        
Optional<Employee> emp =employees.stream()
.collect(Collectors.maxBy(Comparator.comparing(Employee::getSalary)));
        
    9. minBy():
        Find minimum Object.
        return : Optional≤T>
        
Optional<Employee> emp =employees.stream()
.collect(Collectors.minBy(Comparator.comparing(Employee::getSalary)));
        
    10. summarizingInt()//summarizingLong()//summarizingDouble():
        Get statistics
        Returns: IntSummaryStatistics
        
IntSummaryStatistics stats =employees.stream().collect(Collectors.summarizingInt(Employee::getSalary));
        
        Contains: count, sum, average, min, max
    11. mapping():
        Purpose: Transform elements before collecting.
        Returns: Depends on downstream collector.
        
List<String> names =employees.stream().collect(Collectors.mapping(Employee::getName,Collectors.toList()));
        
    12. collectingAndThen():
        Purpose: Perform a final transformation after collecting.
        Returns: Depends on finisher function.
        
List<String> list =stream.collect(Collectors.collectingAndThen(Collectors.toList(),Collections::unmodifiableList));
        
        Returns an immutable list.
    13. groupingBy():
        Purpose: Group elements.
        Returns: Map<K,List<T>>
        
Map<String,List<Employee>> map =employees.stream().collect(Collectors.groupingBy(Employee::getDepartment));
        
    14. groupingBy() with downstream:
        Returns: Map<String, Long>
        
Map<String,Long> map =employees.stream().collect(Collectors.groupingBy(Employee::getDepartment,Collectors.counting()));
        
    15. partitioningBy():
        Purpose: Divide into true/false groups.
        Returns: Map<Boolean,List<T>>
        
Map<Boolean,List<Employee>> map =employees.stream().collect(Collectors.partitioningBy(e -> e.getSalary() > 50000));
        
    16 . partitioningBy() with downstream:
        
Map<Boolean,Long> map =employees.stream().collect(Collectors.partitioningBy(e -> e.getSalary() > 50000,Collectors.counting()));
        
    17. toMap:
        Purpose: Convert to Map.
        Returns: Map<K,V>
        
Map<Integer,String> map =employees.stream().collect(Collectors.toMap(Employee::getId,Employee::getName));
        
    18.toConcurrentMap():
        Returns: ConcurrentMap<K,V>
        Used in parallel streams.
    19. reducing():
        Purpose: Custom reduction.
        Returns: Depends on overload (Optional<T> or T)
        
int total = employees.stream().collect(Collectors.reducing(0,Employee::getSalary,Integer::sum));
        
    20. filtering() (Java 9):
        Purpose: Filter elements within a downstream collector.
        Returns: Depends on downstream collector.
        
Map<String,List<Employee>> map = employees.stream()
.collect(Collectors.groupingBy(Employee::getDepartment,Collectors.filtering(
e -> e.getSalary() > 50000,Collectors.toList())));
        
    21. flatMapping() (Java 9):
        Purpose: Flatten nested streams.
        Returns: Depends on downstream collector.
        
Map<String,Set<String>> map =employees.stream()
.collect(Collectors.groupingBy(Employee::getDepartment,Collectors.flatMapping(
    e -> e.getSkills().stream(),Collectors.toSet())));
        
    22. teeing() (Java 12):
        Purpose: Combine results of two collectors in a single pass.
        Returns: Depends on merger function.
        
double avg =employees.stream().collect(Collectors.teeing(Collectors.summingInt(Employee::getSalary),
Collectors.counting(),(sum,count) -> (double) sum / count));

Quick Interview Cheat Sheet

    | Collector             | Purpose                            | Returns                                        |
    | --------------------- | ---------------------------------- | ---------------------------------------------- |
    | `toList()`            | Collect to List                    | `List<T>`                                      |
    | `toSet()`             | Collect unique elements            | `Set<T>`                                       |
    | `toCollection()`      | Custom collection                  | `Collection<T>`                                |
    | `joining()`           | Join strings                       | `String`                                       |
    | `counting()`          | Count elements                     | `Long`                                         |
    | `summingInt()`        | Sum `int` values                   | `Integer`                                      |
    | `summingLong()`       | Sum `long` values                  | `Long`                                         |
    | `summingDouble()`     | Sum `double` values                | `Double`                                       |
    | `averagingInt()`      | Average `int` values               | `Double`                                       |
    | `averagingLong()`     | Average `long` values              | `Double`                                       |
    | `averagingDouble()`   | Average `double` values            | `Double`                                       |
    | `maxBy()`             | Maximum element                    | `Optional<T>`                                  |
    | `minBy()`             | Minimum element                    | `Optional<T>`                                  |
    | `summarizingInt()`    | Statistics for `int`               | `IntSummaryStatistics`                         |
    | `summarizingLong()`   | Statistics for `long`              | `LongSummaryStatistics`                        |
    | `summarizingDouble()` | Statistics for `double`            | `DoubleSummaryStatistics`                      |
    | `mapping()`           | Transform before collecting        | Depends on downstream collector                |
    | `collectingAndThen()` | Finish with transformation         | Depends on finisher                            |
    | `groupingBy()`        | Group by key                       | `Map<K, List<T>>` (or downstream result)       |
    | `partitioningBy()`    | Split into `true`/`false` groups   | `Map<Boolean, List<T>>` (or downstream result) |
    | `toMap()`             | Convert to `Map`                   | `Map<K, V>`                                    |
    | `toConcurrentMap()`   | Concurrent map                     | `ConcurrentMap<K, V>`                          |
    | `reducing()`          | Custom reduction                   | `T` or `Optional<T>`                           |
    | `filtering()`         | Filter within downstream collector | Depends on downstream collector                |
    | `flatMapping()`       | Flatten nested streams             | Depends on downstream collector                |
    | `teeing()`            | Combine two collectors             | Depends on merger function                     |
