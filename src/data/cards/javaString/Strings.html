String Constant Pool(SCP)

The Java String Constant Pool (SCP) is a dedicated memory region inside the Java Heap that stores unique string literals. Its primary purpose is to optimize memory consumption and improve application performance by preventing the creation of duplicate string objects in memory. Because strings are immutable in Java, the Java Virtual Machine (JVM) can safely share a single string instance among multiple variables. Note :- To Save memory
