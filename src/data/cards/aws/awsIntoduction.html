Introduction to AWS

    AWS (Amazon Web Services) is the world's largest cloud computing platform developed by Amazon.
    Instead of buying physical servers, storage devices, and networking equipment, you rent them from AWS whenever you need them.
    Think of AWS as renting an entire data center over the internet.
    Example:
        Instead of buying:
            10 Servers
            Storage devices 
            Routers
            Database servers
    You simply log in to AWS and create them within minutes.

Cloud Computing

    Delivering computing resources (servers, storage, databases, networking, software, etc.) 
    over the internet instead of owning physical infrastructure.

Pay-As-You-Go Pricing

    This is AWS's biggest advantage.
    Example:
        You create an EC2 server for 10 hours.
    After 10 hours:
        Stop/Delete it.
        Billing stops for compute usage (storage charges may still apply).
    No yearly contracts are required.
    Think of it like an electricity bill:
        Use more → Pay more 
        Use less → Pay less

Benefits of AWS

    1. Cost Effective
        No upfront hardware investment.
    2. Scalability
        Handle:
            10 users, 1,000 users, 10 million users
        by scaling resources up or down.
    3. High Availability
        Applications continue running even if a server fails, 
        thanks to deployment across multiple Availability Zones.
    4. Security
        AWS offers strong security features such as:
            IAM, Encryption,Firewalls,Security Groups,Compliance certifications
    5. Global Reach
        Deploy applications close to your users around the world for lower latency.
    6. Reliability
        AWS infrastructure is designed for high durability and fault tolerance.
